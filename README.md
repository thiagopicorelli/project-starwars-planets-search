# StarWars Planets Search
Trabalho feito na Trybe de uma aplicação de busca por informações de planetas do StarWars, utilizando informações adquiridas da API SWAPI. O projeto foi feito todo utilizando hooks e context para o acesso dos states, e os testes da aplicação foram feitos em Jest.
